# To do list

> This is a web application,  provide user to add daily tasks, check and remove, also it remains in your browsers local storage when you refresh or even close your browser, you will not lose To do list.


## Built With

- Html
- Css
- JavaScript

**Utilities**
- Linters
- Webpack
